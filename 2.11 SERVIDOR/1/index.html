1. URL base del API: https://swapi.co/api/

2. Necesita autenticación: No authentication is required to query and get data

3. Método HTTP que deben usar las peticiones a este API: GET, HEAD, OPTIONS

4. URL para acceder a la info del personaje "Luke Skywalker": https://swapi.co/api/people/1

5. URL para acceder a la info de la película "A New Hope" https://swapi.co/api/films/1

6.a qué otros recursos puedo acceder desde el API además de personajes y pelis: planets / species / vehicles / starships

7. cómo puedo buscar personajes mediante la URL: https://swapi.co/api/people/?search=name

8. cómo puedo hacer que el JSON de una petición se me devuelva traducido a Wookiee: sing the wookiee renderer is easy, just append ?format=wookiee to your urls
